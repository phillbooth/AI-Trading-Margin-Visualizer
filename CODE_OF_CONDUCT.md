# Community Contribution Terms

AIStockBrain is intended to be a practical, respectful, and safety-conscious open source project. By participating in this project, you agree to follow these community terms.

## Be Respectful

- Be kind and constructive when discussing code, trading ideas, bugs, and design choices.
- Critique ideas and implementations, not people.
- Assume good intent, but be clear and direct when something is unsafe, misleading, or technically wrong.
- Do not harass, threaten, insult, discriminate against, or deliberately exclude other contributors.

## Build Responsibly

- Do not submit code intended for market manipulation, fraud, credential theft, evasion, or illegal trading activity.
- Do not commit API keys, secrets, personal data, private market data, or paid datasets that cannot be redistributed.
- Mark risky trading behavior clearly in issues, pull requests, and documentation.
- Keep live-trading features behind explicit safeguards and paper-trading defaults.
- Report security issues privately where possible instead of publishing exploit details in public issues.

## Contribution Expectations

- Keep pull requests focused and explain the reason for the change.
- Include tests or validation notes when behavior changes.
- Document new configuration, risk controls, or external service requirements.
- Respect project maintainers' decisions on scope, safety, and code quality.

## Enforcement

Maintainers may edit, reject, or remove contributions and comments that violate these terms. Repeated or severe violations may result in blocked participation in the project.
